\doxysubsection{Detailed Description}
The class of splitting similar vectors into fixed number of buckets, each bucket is managed by \mbox{\hyperlink{classCANDY_1_1FlatIndex}{Flat\+Index}}. 

\begin{DoxyNote}{Note}
currently single thread 

config parameters
\begin{DoxyItemize}
\item vec\+Dim, the dimension of vectors, default 768, I64
\item initial\+Volume, the initial volume of inline database tensor, default 1000, I64
\item expand\+Step, the step of expanding inline database, default 100, I64
\item number\+Of\+Buckets, the number of titer buckets, default 1, I64, suggest 2$^\wedge$n
\item bucket\+Mode, the mode of assigning buckets, default \textquotesingle{}mean\textquotesingle{}, String, allow the following with its own parameters
\begin{DoxyItemize}
\item \textquotesingle{}mean\textquotesingle{}\+: the bucket is assigned by uniform quantization of the mean, the quantization step is assigned by number\+Of\+Buckets require following parameters
\begin{DoxyItemize}
\item quantization\+Max the max value used for quantization, default 1, Double
\item quantization\+Min the min value used for quantization, default -\/1, Double
\end{DoxyItemize}
\item \textquotesingle{}LSH\+: the bucket is assigned by LSH, and raw LSH encoding will be aggregated according to number\+Of\+Buckets
\begin{DoxyItemize}
\item encode\+Len, the length of LSH encoding, in bytes, default 1, I64
\item metric\+Type, the type of AKNN metric, default L2, String
\item lsh\+Matrix\+Type, the type of lsh matrix, default gaussian, String
\begin{DoxyItemize}
\item gaussian means a N(0,1) LSH matrix
\item random means a random matrix where each value ranges from -\/0.\+5$\sim$0.5
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyNote}

\begin{DoxyItemize}
\item \textquotesingle{}ML\textquotesingle{}\+: the bucket is assigned by maching learning to generate bucket indicies
\begin{DoxyItemize}
\item encode\+Len, the length of LSH encoding, in bytes, default 1, I64
\item metric\+Type, the type of AKNN metric, default L2, String
\item cuda\+Build whether or not use cuda to build model, I64, default 0
\item learning\+Rate the learning rate for training, Double, default 0.\+01
\item hidden\+Layer\+Dim the dimension of hidden layer, I64, default the same as output layer
\item MLTrain\+Batch\+Size the batch size of ML training, I64, default 64
\item MLTrain\+Margin the margin value used in training, Double, default 2$\ast$0.1
\item MLTrain\+Epochs the number of epochs in training, I64, default 10 
\end{DoxyItemize}
