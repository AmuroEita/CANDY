\doxysubsection{Detailed Description}
The class of a \mbox{\hyperlink{classCANDY_1_1YinYangVertex}{Yin\+Yang\+Vertex}}, storing the data in each vertex. 

\begin{DoxyNote}{Note}
now storing each vertex\textquotesingle{}s neighbors, visited number and level, with a pointer to the vector 


\begin{DoxyItemize}
\item yin\+: means this is a summarizing or bridge tensor, not a really data point
\begin{DoxyItemize}
\item yin vertex will only be used for navigation, not output to result
\item yin vertex will be less likely to be deleted, completely changed compared with yang
\item yin vertex can be a summary of multiple tensors
\end{DoxyItemize}
\item yang\+: means the real data point 
\end{DoxyItemize}
\end{DoxyNote}
