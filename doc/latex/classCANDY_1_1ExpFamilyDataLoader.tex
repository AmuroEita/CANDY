\doxysubsection{Detailed Description}
The class to load data from exponential family, i.\+e., poisson, gaussian, exponential and beta. 

\begin{DoxyNote}{Note}
\+:
\begin{DoxyItemize}
\item Must have a global config by \mbox{\hyperlink{classCANDY_1_1ExpFamilyDataLoader_ac1ac02b99154166492e8555fe6774ded}{set\+Config}} 
\end{DoxyItemize}

Default behavior
\begin{DoxyItemize}
\item create
\item call \mbox{\hyperlink{classCANDY_1_1ExpFamilyDataLoader_ac1ac02b99154166492e8555fe6774ded}{set\+Config}}, this function will also generate the tensor A and B correspondingly
\item call \mbox{\hyperlink{classCANDY_1_1ExpFamilyDataLoader_ab1bcb56ad9007e3e57f458c5b8c200e5}{get\+Data}} to get the raw data
\item call \mbox{\hyperlink{classCANDY_1_1ExpFamilyDataLoader_a51ded7079ab2d387bb774837a17a566d}{get\+Query}} to get the query 
\end{DoxyItemize}

parameters of config
\begin{DoxyItemize}
\item vec\+Dim, the dimension of vectors, default 768, I64
\item vec\+Volume, the volume of vectors, default 1000, I64
\item drift\+Position, the position of starting some \textquotesingle{}concept drift\textquotesingle{}, default 0 (no drift), I64
\item parameter\+BetaA, the a parameter in beta distribution, default 2.\+0, double
\item parameter\+BetaB, the b parameter in beta distribution, default 2.\+0, double
\item normalize\+Tensor, whether or not additionally normalize the tensors in L2, 0 (no), I64
\begin{DoxyItemize}
\item drift\+Offset, the offset value of concept drift, default 0.\+5, Double
\item query\+Noise\+Fraction, the fraction of noise in query, default 0, allow 0$\sim$1, Double
\end{DoxyItemize}
\item query\+Size, the size of query, default 10, I64
\item manual\+Change\+Distribution, open this to manually change the distribution, default 0, I64
\item distribution\+Overwrite, the string indicator to manually overwrite the distribution tag, default exponential, String, can be any one of
\begin{DoxyItemize}
\item poisson
\item gaussian
\item exp
\item beta
\end{DoxyItemize}
\item seed, the Exp\+Family seed, default 7758258, I64 
\end{DoxyItemize}

\+: default name tags \char`\"{}\+Exp\+Family\char`\"{}\+: \mbox{\hyperlink{classCANDY_1_1ExpFamilyDataLoader}{Exp\+Family\+Data\+Loader}} 
\end{DoxyNote}
