\doxysubsection{Detailed Description}
The basic class to generate time stamps. 

\begin{DoxyNote}{Note}
require configs\+:
\begin{DoxyItemize}
\item event\+Rate\+Tps I64 The real-\/world rate of spawn event, in Tuples/s
\item streaming\+Tuple\+Cnt I64 The number of \char`\"{}streaming tuples\char`\"{}, can be set to the \#rows or \#cols of a matrix
\item time\+Stamper\+\_\+zipf\+Event, I64, whether or not using the zipf for event rate, default 0
\item time\+Stamper\+\_\+zipf\+Event\+Factor, Double, the zpf factor for event rate, default 0.\+1, should be 0$\sim$1
\item static\+Data\+Set, I64, 0 , whether or not treat a dataset as static 
\end{DoxyItemize}

Default behavior
\begin{DoxyItemize}
\item create
\item call \mbox{\hyperlink{classINTELLI_1_1IntelliTimeStampGenerator_ae0823d197bd017cd93aba76bfd6b0a97}{set\+Config}} to generate the timestamp under instructions
\item call \mbox{\hyperlink{classINTELLI_1_1IntelliTimeStampGenerator_acf86dff87f8eee751aec0dd5d00a2ecf}{get\+Time\+Stamps}} to get the timestamp 
\end{DoxyItemize}
\end{DoxyNote}
