\doxysubsection{Detailed Description}
The top entity to trace current, average and maximum memory foot print. 

\begin{DoxyNote}{Note}
The default unit is KB, will use Linux timer to keep sampling memory usage 

usage
\begin{DoxyItemize}
\item create a class
\item call INTELLI\+::\+Memory\+Tracker\+::set\+Active\+Instance(\&xxx) to register this to linux timer
\item call \mbox{\hyperlink{classINTELLI_1_1MemoryTracker_aac84e16cfbdc70a0ecf5ad32fd878276}{start}} to start the sampling
\item call end to end the sampling
\item call \mbox{\hyperlink{classINTELLI_1_1MemoryTracker_a028cce03f4affbaf5f7637ee3b7cb03c}{get\+Avg\+Mem}}, \mbox{\hyperlink{classINTELLI_1_1MemoryTracker_acf3d6f332d8cd34914f589711cc176f5}{get\+Max\+Mem}}, or \mbox{\hyperlink{classINTELLI_1_1MemoryTracker_acbc820884c8c940ad75106d27515d348}{get\+Cur\+Mem}} to get the result, \mbox{\hyperlink{classINTELLI_1_1MemoryTracker_acbc820884c8c940ad75106d27515d348}{get\+Cur\+Mem}} is a instant function rather than reporting the sampled results 
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Never use multiple instance of INTELLI\+::\+Memory\+Tracker\+::set\+Active\+Instance(\&xxx) 
\end{DoxyWarning}
