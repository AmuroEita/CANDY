\doxysubsection{Detailed Description}
The class to load zipf data. 

\begin{DoxyNote}{Note}
\+:
\begin{DoxyItemize}
\item Must have a global config by \mbox{\hyperlink{classCANDY_1_1ZipfDataLoader_ab46e5be2d5b3068027c9fbb7250ae2f2}{set\+Config}} 
\end{DoxyItemize}

Default behavior
\begin{DoxyItemize}
\item create
\item call \mbox{\hyperlink{classCANDY_1_1ZipfDataLoader_ab46e5be2d5b3068027c9fbb7250ae2f2}{set\+Config}}, this function will also generate the tensor A and B correspondingly
\item call \mbox{\hyperlink{classCANDY_1_1ZipfDataLoader_a4df081f7d1b0efcd75153ec228ee3451}{get\+Data}} to get the raw data
\item call \mbox{\hyperlink{classCANDY_1_1ZipfDataLoader_a43117858967fbb5921eee16325240ca5}{get\+Query}} to get the query 
\end{DoxyItemize}

parameters of config
\begin{DoxyItemize}
\item vec\+Dim, the dimension of vectors, default 768, I64
\item vec\+Volume, the volume of vectors, default 1000, I64
\item normalize\+Tensor, whether or not normalize the tensors in L2, 1 (yes), I64
\item \char`\"{}zipf\+Alpha\char`\"{} The zipf factor for, Double, 0-\/highly skewed value. 1-\/ uniform dist.
\item drift\+Position, the position of starting some \textquotesingle{}concept drift\textquotesingle{}, default 0 (no drift), I64
\begin{DoxyItemize}
\item drift\+Offset, the offset value of concept drift, default 0.\+5, Double
\item query\+Noise\+Fraction, the fraction of noise in query, default 0, allow 0$\sim$1, Double
\end{DoxyItemize}
\item query\+Size, the size of query, default 10, I64
\item seed, the Zipf seed, default 7758258, I64 
\end{DoxyItemize}

\end{DoxyNote}
