\doxysubsection{Detailed Description}
The class of ranom data loader,. 

\begin{DoxyNote}{Note}
\+:
\begin{DoxyItemize}
\item Must have a global config by \mbox{\hyperlink{classCANDY_1_1RandomDataLoader_a04300ab31584bc096aeb0011ac1f44fe}{set\+Config}} 
\end{DoxyItemize}

Default behavior
\begin{DoxyItemize}
\item create
\item call \mbox{\hyperlink{classCANDY_1_1RandomDataLoader_a04300ab31584bc096aeb0011ac1f44fe}{set\+Config}}, this function will also generate the tensor A and B correspondingly
\item call \mbox{\hyperlink{classCANDY_1_1RandomDataLoader_a9cddb70dded05202c564035456890b0d}{get\+Data}} to get the raw data
\item call \mbox{\hyperlink{classCANDY_1_1RandomDataLoader_a2b4c9f41e32bcebc820bdd494a35d516}{get\+Query}} to get the query 
\end{DoxyItemize}

parameters of config
\begin{DoxyItemize}
\item vec\+Dim, the dimension of vectors, default 768, I64
\item vec\+Volume, the volume of vectors, default 1000, I64
\item drift\+Position, the position of starting some \textquotesingle{}concept drift\textquotesingle{}, default 0 (no drift), I64
\begin{DoxyItemize}
\item drift\+Offset, the offset value of concept drift, default 0.\+5, Double
\item query\+Noise\+Fraction, the fraction of noise in query, default 0, allow 0$\sim$1, Double
\end{DoxyItemize}
\item query\+Size, the size of query, default 10, I64
\item seed, the random seed, default 7758258, I64 
\end{DoxyItemize}

\+: default name tags \char`\"{}random\char`\"{}\+: \mbox{\hyperlink{classCANDY_1_1RandomDataLoader}{Random\+Data\+Loader}} 
\end{DoxyNote}
