\doxysubsection{Detailed Description}

\begin{DoxyNote}{Note}
default behaviors\+:
\begin{DoxyItemize}
\item create
\item call \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_af0f2d9ecca1d329f41f5fa7e0b757a0b}{set\+Config()}} to config this meter
\item (optional) call \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_a2dac7c7167003ff6fcd2e8661d0d8463}{test\+Static\+Power()}} to automatically test the static power of a device or \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_a6591be89d8c5c7cf4a93226aa14cff71}{set\+Static\+Power}} to manually set the static power, if you want to exclude it

\item (run your program)
\item call \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_aaa9a1688e3a9850a425079309e7ee15b}{stop\+Meter()}} to stop measurement
\item call \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_a510602e0efac1cff564dff6db556b58a}{get\+E()}}, \mbox{\hyperlink{classDIVERSE__METER_1_1AbstractMeter_adb52ca60dfd43c83f0f7ba60ac977357}{get\+Peak()}}, etc to get the measurement resluts
